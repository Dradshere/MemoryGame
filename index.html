How does this game work?

Its simple, Whenever a cell flashes red remember what number was in that cell.
But when the same cell flashes red again, you can forget the first one.

In level 2 you'll find that theres another question:
"The sum of every number shown on screen:"
There you'll need to input every number you have seen that round.
Even if that number would have been flashed again.
